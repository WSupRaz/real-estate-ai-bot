# 🏡 Real Estate AI Assistant

An AI-powered real estate chatbot that helps users discover properties, get recommendations, and interact like a real estate agent.

---

## 🚀 Live Demo
(Coming soon)

---

## ✨ Features

- 🤖 AI-powered conversational chatbot  
- 🏠 Smart property recommendations  
- ⚡ Fast semantic search using FAISS  
- 💬 Real-time chat interface  
- 📊 Context-aware responses using embeddings  
- 🔐 Secure API handling (no secrets exposed)  

---

## 🧠 Tech Stack

- **Frontend/UI**: Streamlit  
- **AI Model**: OpenRouter (Mistral-7B)  
- **Embeddings**: Sentence Transformers  
- **Vector Search**: FAISS  
- **Backend**: Python

## ⚙️ Installation & Setup

### 1. Clone the repository


git clone https://github.com/WSupRaz/real-estate-ai-bot.git

cd real-estate-ai-bot


---

### 2. Install dependencies


pip install -r requirements.txt


---

### 3. Add API Key

Create a `.env` file:


OPENROUTER_API_KEY=your_api_key


---

### 4. Run the app


streamlit run app.py
